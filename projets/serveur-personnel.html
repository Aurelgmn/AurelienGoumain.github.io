Serveur Personnel
Automatisations & Self-Hosting

Hébergement de services sur Raspberry Pi via Docker : workflows d'automatisation n8n, scripts Python, intégration d'APIs REST et administration réseau complète.

Infrastructure

Raspberry Pi · Docker · Linux

Mise en place d'un serveur domestique sur Raspberry Pi 4 sous Raspberry Pi OS (Debian). Déploiement et orchestration de plusieurs services via Docker Compose : isolation des services, gestion des volumes persistants, des réseaux internes et des variables d'environnement sensibles.

Configuration d'un reverse proxy (Nginx) pour exposer les services via HTTPS avec certificats Let's Encrypt, et gestion des règles de pare-feu.

Raspberry PiDocker ComposeNginxLet's Encrypt

Automatisations n8n

Workflow automation

Déploiement de n8n (outil d'automatisation open-source) pour construire des workflows sans code connectant différents services entre eux. Exemples de workflows : notifications automatiques, synchronisation de données entre APIs, alertes conditionnelles, pipelines de traitement de données déclenchés par webhooks.

Maîtrise de la logique de branchement, des transformations JSON et de la gestion des erreurs.

n8nWebhooksWorkflowsJSON

Scripts Python & APIs

API REST · HTTP · Automatisation

Développement de scripts Python pour des automatisations plus complexes nécessitant de la logique métier : consommation d'APIs REST avec la librairie requests, parsing et transformation de données JSON, authentification OAuth2 et gestion des tokens.

Les scripts tournent sous forme de tâches planifiées (cron) ou sont déclenchés via des webhooks exposés par n8n.

Python requestsOAuth2CronREST API

Administration réseau

DNS · Ports · Sécurité

Configuration de l'accès distant sécurisé : redirection de ports sur le routeur, configuration DNS (domaine custom), accès SSH sécurisé (clés, fail2ban). Monitoring des services avec alertes en cas d'arrêt inopiné.

Ce projet m'a permis de développer une compréhension concrète du réseau TCP/IP, des protocoles HTTP/HTTPS et des bonnes pratiques de sécurité serveur.

SSHDNSTCP/IPSécurité
